## Overview

X posting lets you edit selected gallery or favorites photos (up to 4) in a post modal and publish them with an optional camera-info frame. Text presets are managed in Settings.

X linking and posting are under **staged rollout**. Availability depends on release status, plan, account state, and environment.

## How to open

1. Sign in on the VSA website and link X there if needed
2. Check login state and X posting status on the app Account screen
3. Select photos in Gallery/Favorites and open **Post to X** or X post mode
4. Manage presets under **Settings > X Post**

## Main operations

### Post modal

Enter text, apply a preset, review photos, and post.

![X post modal](../../assets/screenshots/xpost-modal.webp)

### Camera info frame

Preview and include a camera-info frame with the post when desired.

![Camera info frame](../../assets/screenshots/xpost-camera-frame.webp)

### Settings presets

In the X Post settings panel, add, edit, or delete text-template presets. There is no in-app X OAuth flow here.

![X post settings](../../assets/screenshots/settings-xpost.webp)

### Account X status

The Account screen shows X posting status such as plan and remaining monthly posts. Unlink/manage X on the website account page.

## Notes

- The old settings-panel OAuth auth flow is removed. Website login and X linking are the source of truth
- Outside rollout, post buttons or status may be hidden
- Limits and pricing follow in-app display and the [Terms of Service](terms.md)
