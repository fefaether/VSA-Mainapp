## Overview

Favorites mark photos with a star for quick local browsing. When cloud save is available, favorites can also be backed up (staged rollout).

## How to open

1. Click the star icon in the gallery to favorite a photo
2. Open the **Favorites** tab
3. Check cloud-related options on the Account screen or in Settings > Favorites

## Main operations

### Favorites list

The Favorites tab lists starred photos. Detail view and selection work like the gallery.

![Favorites overview](../../assets/screenshots/favorites-overview.webp)

### Cloud sync (rollout)

When available, review cloud save status. If it is missing, your environment is outside the current rollout.

![Cloud sync](../../assets/screenshots/favorites-cloud-sync.webp)

## Notes

- Local favorites work without an account
- Cloud save depends on release status, plan, and account state
- Migration and management in Settings are covered in the [Settings Guide](settings-guide.md)
