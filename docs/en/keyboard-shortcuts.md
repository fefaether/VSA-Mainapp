## Overview

Keyboard shortcuts are listed and customized in Settings > Shortcut. Targets include window size, fullscreen, and gallery/retouch zoom.

## How to open

1. Open **Settings > Shortcut**
2. Select the row you want to change
3. Press the new key combination and save

## Main operations

### Shortcut list

Review current bindings in the settings panel. Common defaults:

| Action | Default |
|--------|---------|
| Cycle window size | `F10` |
| Toggle fullscreen | `F11` |
| Gallery zoom in | `Ctrl` + `=` |
| Gallery zoom out | `Ctrl` + `-` |

![Shortcut settings](../../assets/screenshots/settings-shortcut.webp)

### Customize

Click a row, wait for key capture, then register a new combination. You can also reset to defaults from the panel.

![Customize shortcuts](../../assets/screenshots/shortcut-customize.webp)

## Notes

- Shortcuts may conflict with OS or other apps
- Some shortcuts only work on specific screens (gallery vs retouch)
- If a shortcut seems ignored, make sure focus is not in a text field
