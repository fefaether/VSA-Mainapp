# レタッチ機能ガイド

[🏠 ドキュメントトップ](../index.md) | [⚖️ 利用規約](./terms.md) | [🔒 プライバシーポリシー](./privacy.md)

---


レタッチ機能を使用して、撮影した写真を直接アプリケーション内で編集できます。明度、コントラスト、色温度、トーンカーブなど、多数の調整が可能です。

## レタッチエディタの開き方

ギャラリーから写真を選択し、「レタッチ」ボタンをクリックしてレタッチエディタを起動します。

## 画面構成

レタッチエディタは3つの領域で構成されています：

### 左側領域（メイン編集エリア）

- **ズームコントロール** — 写真の表示倍率を調整
- **プレビューキャンバス** — 現在の編集結果をリアルタイムで表示
- **フィルムストリップ** — 最近の写真を一覧表示して素早く切り替え可能

### 右側パネル（パラメータ調整）

各種編集パラメータをスライダーまたは数値入力で調整できます。

## 編集パラメータ

### 基本パラメータ

| パラメータ | 説明 | 範囲 |
|-----------|------|------|
| **明度（Brightness）** | 写真全体の明るさを調整 | -100% ～ +100% |
| **コントラスト（Contrast）** | 明暗の差を調整して、メリハリのある写真に | -100% ～ +100% |
| **彩度（Saturation）** | 色の鮮やかさを調整 | -100% ～ +100% |
| **色温度（Color Temperature）** | 写真の色合い（暖色・寒色）を調整 | 2000K ～ 10000K |
| **ハイライト（Highlights）** | 明るい領域（白い部分）を調整 | -100% ～ +100% |
| **シャドウ（Shadows）** | 暗い領域（黒い部分）を調整 | -100% ～ +100% |

### パラメータの調整方法

#### スライダーで調整

各パラメータの横にあるスライダーをドラッグして値を調整します。

#### 数値で直接入力

右側の数値欄をクリックして、直接数値またはパーセンテージを入力します。

- `50` または `50%` で +50%
- `-30` または `-30%` で -30%

#### リセット（デフォルト値に戻す）

各パラメータ行の右端にあるリセットボタン（↻アイコン）をクリックして、デフォルト値に戻します。

## トーンカーブエディタ

より細かい色調補正を行うために、トーンカーブエディタを使用できます。

### トーンカーブの基本

トーンカーブは、入力値（横軸）と出力値（縦軸）の関係を曲線で表します。

- **RGB**：全ての色チャンネルを同時に調整
- **R（赤）**：赤チャンネルのみを調整
- **G（緑）**：緑チャンネルのみを調整
- **B（青）**：青チャンネルのみを調整

### トーンカーブの操作方法

1. 左側パネルにある「トーンカーブ」セクションを開く
2. チャンネル選択ボタン（RGB/R/G/B）をクリックして調整対象を選択
3. 曲線上の任意の場所をクリックしてポイントを追加
4. ポイントをドラッグして曲線を調整
5. 不要なポイントは削除アイコンをクリックして削除

### トーンカーブ調整例

**シャドウを持ち上げる**：左下の曲線を上に持ち上げて、暗い部分を明るくする

**ハイライトを抑える**：右上の曲線を下に下げて、明るい部分の露出をコントロール

**コントラストを強化**：曲線をS字型にして、影と光の差を強調

## ズーム操作

### ズームボタン

プレビューキャンバス上部のズームコントロールを使用：

- **ズームイン（+）** — 詳細を拡大表示
- **ズームアウト（-）** — 全体を小さく表示
- **フィットウィンドウ** — 写真全体を画面に収める（デフォルト）

### キーボードショートカット

| 操作 | ショートカット |
|------|----------------|
| ズームイン | `Ctrl` + `=` |
| ズームアウト | `Ctrl` + `-` |
| フィットウィンドウ | `Ctrl` + `0` |

## フィルムストリップ

下部のフィルムストリップで、最近の100枚の写真を一覧表示します。

### フィルムストリップの操作

- **写真選択** — フィルムストリップ内の写真をクリックして即座に切り替え
- **スクロール** — マウスホイールまたはスクロールバーで左右にスクロール

## ツールバーボタン

### 編集操作

| ボタン | 機能 |
|--------|------|
| **戻す（Undo）** | 最後の操作を取り消す（`Ctrl` + `Z`） |
| **やり直す（Redo）** | 取り消した操作をもう一度実行（`Ctrl` + `Shift` + `Z`） |
| **リセット** | すべてのパラメータをデフォルト値に戻す |

### 適用・保存

| ボタン | 機能 |
|--------|------|
| **適用（Apply）** | 編集結果を写真に適用して保存 |
| **キャンセル（Cancel）** | 編集内容を破棄してレタッチを終了 |
| **エクスポート（Export）** | 編集結果を新しいファイルとしてエクスポート |

### エクスポート機能

**エクスポート**ボタンをクリックすると、以下のオプションで保存できます：

- **ファイル形式** — PNG、JPEGなど対応形式を選択
- **ファイル名** — 保存先と新しいファイル名を指定
- **品質** — JPEG保存時の圧縮品質を調整

## よくある操作

### 暗い写真を明るくする

1. **明度**スライダーを右に移動して全体を明るくする
2. 必要に応じて**ハイライト**を調整して白飛びを防止

### 写真の色を暖かくする

1. **色温度**スライダーを右に移動（高い値 = 暖色）
2. 調整結果をプレビューで確認

### 色の鮮やかさを上げる

**彩度**スライダーを右に移動して、色をより鮮やかにする

### 影と光のバランスを整える

1. **シャドウ**で暗い部分を調整
2. **ハイライト**で明るい部分を調整
3. 全体のバランスを確認しながら微調整

## 注意事項

- **作業の自動保存** — エディタを閉じる際に、「適用」ボタンを押さない限り、編集内容は保存されません
- **元のファイルへの影響** — 「適用」で保存した場合、元の写真が上書きされます。重要な写真の場合は、エクスポート機能で別名保存することをお勧めします
- **大きなファイルの処理** — 高解像度の写真は処理に時間がかかる場合があります
- **メモリ使用量** — 複数の写真を同時にレタッチする場合、メモリを消費します
