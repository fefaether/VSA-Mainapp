# クラウド・アカウント連携（旧 Google 連携）

[🏠 ドキュメントトップ](../index.md) | [⚖️ 利用規約](./terms.md) | [🔒 プライバシーポリシー](./privacy.md)

VSAでは、クラウド保存、共有リンク、X投稿などのアカウント連携機能を提供・拡張しています。一部の環境では外部サービス（例: Google OAuth、X OAuth）による認証が利用される場合がありますが、**Google ドライブや Google フォトへの一般的な公開提供を本ドキュメントでは前提としません**。

## 機能概要（ロールアウト中）

次の機能は**段階的なロールアウト**の対象です。利用可否は公開状況、プラン、アカウント状態、利用環境により異なります。

- VSAアカウントでのサインイン
- クラウド保存（お気に入り等の任意バックアップ）
- 共有リンクの発行
- Xへの投稿連携
- 複数デバイス間での同期（提供される場合）

アプリに該当UIが表示されない場合、その機能はまだ利用できない可能性があります。

## 認証の一般的な流れ（表示される場合）

外部アカウント連携が有効な環境では、おおむね次の流れになります。

1. 設定またはアカウント画面を開く
2. 連携したいサービスまたはVSAアカウントの案内に従う
3. ブラウザで認証し、アプリに戻る
4. 連携状態とプランを確認する

要求される権限の範囲はサービスとロールアウト段階により異なります。詳細は認証画面と[プライバシーポリシー](./privacy.md)を確認してください。

## プライバシーとセキュリティ

- ローカルに保存されるデータ（お気に入り、設定等）はお使いのPC上に保持されます
- クラウド保存や共有を利用する場合、送信されるデータの種類と保管先はプライバシーポリシーに記載されています
- OAuth等の認証では、可能な限り必要最小限の権限のみを要求します
- 連携の解除はアカウント画面または各サービスの設定から行える場合があります

## トラブルシューティング（共通）

### 連携メニューが表示されない

- ロールアウト対象外の環境である可能性があります
- アプリを最新版に更新し、アカウント状態を確認してください

### 認証が完了しない

1. インターネット接続を確認する
2. ブラウザのポップアップブロックを解除する
3. 別のブラウザで再試行する
4. VSAを再起動する

### 同期やアップロードが失敗する

1. プラン、月間上限、無料枠をアプリ内表示で確認する
2. ネットワークが安定しているか確認する
3. 重要なデータはローカルバックアップを維持する
4. 解決しない場合は[トラブルシューティング](troubleshooting.md)と[FAQ](faq.md)を参照する
