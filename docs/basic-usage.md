# 基本的な使い方

最終更新日: 2026年4月24日

VSA（Virtual Snap Archive）は、写真をローカルで整理し、撮影時の文脈やカメラ情報を管理するためのアプリです。

## 1. 初期設定

1. VSAを起動します。
2. 写真の取り込み対象フォルダを指定します。
3. 必要に応じて書き出し先、表示設定、圧縮設定を調整します。

VSAは、ユーザーが監視フォルダや書き出し先を自由に変更できるよう、広めのローカルファイルアクセス権限を持っています。

## 2. 写真管理

- ギャラリーで写真を一覧表示できます。
- 写真詳細で撮影日時、ワールド、ユーザー、カメラ情報などを確認できます。
- ワールド名、ユーザー名、日付などで写真を探せます。
- お気に入りやエクスポートで共有準備ができます。

## 3. JPEG XL圧縮

JPEG XL圧縮を使うと、画質を保ちながら写真の容量を削減できます。重要な写真は、圧縮前後にかかわらず別の場所にもバックアップすることを推奨します。

## 4. クラウド保存

クラウド保存を有効にすると、対象の写真とメタデータがVSAウェブサイトAPIおよびCloudflare R2等へ送信されます。無料枠を超える利用では料金が発生する場合があります。

クラウド保存を使わない場合、写真管理は主にローカルPC内で完結します。

## 5. X投稿

X投稿を使うには、VSAアカウントとXアカウント連携が必要です。

投稿時は、投稿文、画像、月間上限、残り回数を確認したうえで、ユーザーが毎回明示的に投稿操作を行います。OAuth連携だけで自動投稿されることはありません。

## 6. 外部通信

以下の機能では外部通信が発生します。

- VSAアカウント認証
- クラウド保存
- X投稿
- Cloudflare Turnstile
- アップデート確認
- サーバーステータス表示

詳細はプライバシーポリシーを確認してください。
