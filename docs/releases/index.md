## v0.6.0

### 新機能

- `EXIFスタジオ`ベータを追加し、カメラ情報フレームや透かし付き画像の書き出し、プリセット保存、X投稿向けの出力をまとめて扱えるようにしました。
- ゲーム関連設定に`VRChatイベントタイムライン`を追加し、ワールド入退室やユーザー参加・退出の履歴を期間指定で確認できるようにしました。
  サイドバーの`ゲーム関連設定`を開き、タイトルバーの`タイムマシーン`タブからアクセスできます。
- ゲーム関連設定に`VRChatサーバーステータス`監視を追加し、現在の障害状況や項目ごとの状態を確認しやすくしました。
- 自動圧縮アーカイブ機能を追加し、圧縮済みファイルの管理、復元、初回セットアップを進めやすくしました。

### 改善

- ギャラリー、お気に入り、写真詳細の操作を見直し、右クリック共有、長押し選択、サイドバー導線を改善しました。
- 初回セットアップ、チュートリアル、キーボード操作、ローカル画像表示を見直し、使い始めの詰まりを減らしました。
- `EXIFスタジオ`とレタッチ書き出しの保存先選択をわかりやすくし、書き出し中の進行表示やモーダル挙動を改善しました。
- OSC/OSCQueryの自動検出と接続安定性を改善し、開発者モードのOSCモニターでは任意アドレスへショートカットを割り当てて送信できるようにしました。
- 通知設定を見直し、OS通知とトースト通知を個別に切り替えやすくしました。
- タイムラインの日付範囲フィルター、サーバーステータス通知、シャッタースピード表示など、VRChat関連の表示精度を改善しました。

### バグ修正

- ローカル画像プレビュー、初回サムネイル、欠損画像時の表示崩れを修正しました。
- レタッチとEXIF書き出しで発生していた日本語フォント欠落、プレビュー差分、レイアウト崩れ、角丸表示の問題を修正しました。
- QRコードURLの検証、VRChatログ解析、ログ保持設定、カメラ判定競合など、安定性に関わる問題を修正しました。

### その他

- 利用規約、プライバシーポリシー、設定画面の外部リンク案内を更新し、公開ドキュメントの正本へ統一しました。
- リリース前チェックと配布設定を見直し、更新確認やログイン導線がリリース版で誤った接続先を向かないようにしました。
- 連携先検証、ファイルアクセス境界、取り込みとアップロード周辺の安全性を強化しました。

### 一部ユーザー向け機能

- `VSAアカウント`のベータ導線を整理し、ログイン状態、プラン、連携状況をアプリ内で確認しやすくしました。
- お気に入りのクラウド同期、共有リンク生成、X投稿まわりの導線を強化しました。利用できる機能は公開状況、プラン、アカウント状態、利用環境により異なります。
- VRC+外ふち写真のクラウド除外に対応し、除外状態や関連設定をアプリ内で確認できるようにしました。
- クラウド同期のページ上限、再送、再起動復旧、混雑時の再試行まわりの不具合を修正しました。
- X投稿モーダルの未認証表示、小画面での見切れ、ログイン復帰時の挙動を修正しました。

### 既知の注意点

- `VSAアカウント`、クラウド保存、共有リンク、X投稿、`EXIFスタジオ`は、段階的公開またはベータ扱いの機能を含みます。利用可否は公開状況、プラン、アカウント状態、利用環境により異なります。
- ローカル写真管理の基本機能は、これまでどおり無料で利用できます。
- 重要な写真やデータは、VSAの管理フォルダとは別の場所にもバックアップしてください。

## リリースページ本文の構成（推奨）

GitHub Releases の本文は、以下の構成で記載します。

1. **新機能**
2. **改善**
3. **バグ修正**
4. **既知の問題**（該当がある場合のみ）
5. **その他**（依存関係更新・運用変更など）

### 書き方の基準

- 1項目=1行で簡潔に書く（ユーザーが一目で理解できる粒度）。
- まず「何ができるようになったか」を書き、その後に補足理由を書き足す。
- 技術用語は必要最小限にし、ユーザー視点の効果を優先する。
- 機能名が紛らわしい場合は、UI上の名称に合わせる。
