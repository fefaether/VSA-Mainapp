## 概要

VRChat連携のためのフォルダ監視やOSC通信、タイムマシーンなどをまとめて設定できるパネルです。撮影検知とメタデータ取得の土台になります。

## ユーザーに見えるポイント

- 監視のオン/オフを切り替えできる
- スクリーンショット/メタデータ/ログのフォルダ設定を行える
- OSCポート設定によりカメラ情報取得が安定する
- タイムマシーンで過去インスタンス履歴を確認できる

## 操作・導線

- サイドバーの **「ゲーム側設定」** からアクセス
- タブ構成: 画面設定 / タイムマシーン / サーバー
- サーバーステータスは別ページで詳細化

## リリース本文で触れるべき点

- フォルダ監視/メタデータ取得の設定導線
- OSC連携の安定化やポート設定の明確化
- タイムマシーンの可視化・再参加導線
