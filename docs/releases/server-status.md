## 概要

VRChatのサーバーステータスを確認できる機能です。メンテナンスや障害状況を把握できます。

## ユーザーに見えるポイント

- 全体ステータス（正常/劣化/障害/メンテ）を確認できる
- コンポーネント別の状態が一覧表示される
- 定期更新で最新状態を追従できる

## 操作・導線

- **ゲーム側設定** の **サーバータブ** からアクセス

## リリース本文で触れるべき点

- サーバー稼働状況がアプリ内で確認できる
- 障害時に状態が可視化される
