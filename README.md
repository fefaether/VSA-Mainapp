## VSAって何？

Virtual Snap Archive（VSA）は、写真を整理し、撮影時の文脈やカメラ情報をあとから確認できるようにするアプリです。

- 写真がフォルダに散乱して整理できない
- どこで撮ったか、誰と撮ったか思い出せない
- 大量のスクショでストレージがパンパン

VSAは、ローカル管理を基本にしつつ、ユーザーが選択した場合のみVSAアカウント、クラウド保存、X投稿などの外部サービス連携を利用します。

## インストール方法

### システム要件

- **OS**: Windows 10/11 (64bit推奨)
- **メモリ**: 4GB以上推奨（2GB最小）
- **ディスク**: 500MB以上の空き容量（圧縮機能使用時は別途容量必要）
- **画面**: 1920×1080以上推奨

### インストール手順

1. [**ダウンロードページ**](https://github.com/fefaether/VSA-Mainapp-Release/releases/)へ移動
2. 最新版の`VSA-Setup-vX.X.X.exe`をダウンロード
3. ダウンロードしたファイルを実行
4. 画面の指示に従ってインストール
5. デスクトップのアイコンからVSAを起動！

## できること

### VSA - 写真管理アプリ

| 機能 | 説明 |
|------|------|
| **ギャラリー表示** | 撮った写真を一覧で見やすく表示 |
| **自動取り込み** | VRChatの写真を自動で取り込み |
| **撮影情報を自動保存** | ワールド名、撮影日時などを自動記録 |
| **写真を圧縮** | 高画質のまま容量を30-50%削減（JPEG XL対応） |
| **お気に入り機能** | 大切な写真にマークを付けて管理 |
| **トラッシュ機能** | 削除した写真を復元可能 |
| **ワールド検索** | 特定のワールドで撮った写真を検索 |
| **フォルダ一括取り込み** | VRChat以外のフォルダからも取り込み可能 |
| **テーマ切替** | ダーク/ライトモード選択可能 |
| **OSC連携** | VRChat内からアプリ操作が可能 |
| **画像レタッチ** | 明度・コントラスト・色温度・トーンカーブ調整 |
| **複数言語対応** | 日本語・英語ほか多言語対応 |

### VDI - 写真ビューワ

| 機能 | 説明 | 対応状況 |
|------|------|---------|
| **拡大・縮小** | 細部まで確認できる高精度ズーム | ✅ |
| **構図ガイド** | 三分割法などの構図ガイド表示 | ✅ |
| **ピント確認** | ピントが合っている場所をハイライト | ⚠️ 一部不安定 |
| **ヒストグラム** | 写真の明るさ分布を確認 | ✅ |
| **グリッド表示** | グリッドオーバーレイで構図確認 | ✅ |
| **画像情報表示** | 解像度・ファイルサイズ等を表示 | ✅ |

**VDI対応状況**: 基本機能は動作します。ピント確認機能など一部で処理が不安定な場合があります。今後のアップデートで改善予定です。

## 使い方

### はじめての設定

1. **写真フォルダを設定する**
   - サイドバーの「ゲーム側設定」をクリック
   - 「フォルダを選択」でVRChatの写真保存先を選ぶ
   - 通常は `C:\Users\あなたの名前\Pictures\VRChat`

2. **監視をスタート**
   - 「監視開始」ボタンをクリック
   - これで新しい写真が自動で取り込まれます！

3. **ギャラリーで確認**
   - サイドバーの「ギャラリー」をクリック
   - 取り込まれた写真が一覧表示されます

### よく使う操作

| やりたいこと | 方法 |
|--------------|------|
| 写真を大きく見る | サムネイルをクリック |
| お気に入り登録 | お気に入りボタンをクリック |
| 写真を圧縮 | 設定からJXL圧縮を有効化 |

## よくある質問

### 写真が表示されない

1. フォルダが正しく設定されているか確認
2. 「監視開始」ボタンを押したか確認
3. アプリを再起動してみる
4. VSAが読み取り権限を持っているか確認

### 容量を節約したい

設定画面でJXL圧縮を有効にすると、写真の容量を30-50%削減できます。
画質はほとんど変わりません！

### VRChatプレイ中に動作が重くなる？

大丈夫！VRChatが起動中は自動的に処理を一時停止します。

### 圧縮中にアプリがハングしているようにみえる

JXL圧縮は大きなファイルほど時間がかかります（数秒～数十秒）。
処理完了まで少しお待ちください。複数ファイルは順番に処理されます。

### OSC連携を使いたい

VRChatで「開発者設定」を有効にしてOSCを許可してください。
VSAの設定から「OSC連携」を有効にすれば、VRChat内から操作可能です。

### ワールド名が自動で保存されない

VRChat側でワールド情報がログに記録されていない可能性があります。
VRChat設定を確認するか、手動でワールド情報を入力することもできます。

### アップデート方法は？

新しいバージョンがリリースされると、アプリ起動時に自動通知されます。
設定から「今すぐアップデート」をクリックするか、最新版をダウンロードして上書きインストールしてください。
